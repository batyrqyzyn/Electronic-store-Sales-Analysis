# Electronics Store Sales Analysis

## Project Overview

This project demonstrates the complete data analysis process using PostgreSQL, SQL, Excel, and Power BI.

The objective was to design a relational database, analyze sales data using SQL, perform additional analysis in Excel, and create a Power BI dashboard for data visualization.

## Technologies

- PostgreSQL
- SQL
- Excel
- Power BI
- Git
- GitHub

## Database Structure

The project consists of four related tables:

- Customers
- Products
- Orders
- Orders_Details

## Project Tasks

- Designed a relational database
- Created tables and relationships
- Inserted sample data
- Wrote SQL queries for data analysis
- Performed data analysis in Excel
- Built a Power BI dashboard

## SQL Analysis

The project includes queries using:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- JOIN
- COUNT
- SUM
- AVG
- MIN
- MAX

## Excel Analysis

Excel was used for:

- Pivot Tables
- XLOOKUP
- SUMIF
- COUNTIF
- Charts

## Power BI Dashboard

The dashboard presents:

- Total Revenue
- Revenue Trend by Order Date
- Product Filter
- Interactive Sales Visualization
